## Why this folder is empty (on purpose)

The training module must be built from **observed behavior**, not imagined curriculum:

- Which discovery calls actually die at minute fifteen  
- Which silence drills break which operators  
- Which kill decisions feel uncomfortable but correct  
- Which candidates enforce without asking Rachel  

Building slides before Week 4 internal certification produces training that teaches improvisation with better graphics.
